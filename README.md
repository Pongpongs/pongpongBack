how to launch

1. reinstall venv
git clone (clone_name)

cd pongpongBack

rm -rf myprojectvenv

python3 -m venv myprojectvenv

2. activate venv
. myprojectvenv/bin/activate

3. install modules
pip install -r requirements.txt

4. launch server
cd WebService

bash execute_server.sh
